# **Match Patterns**

The match patterns in Rust.

Content:

- Match syntex:
    -   match branch;
    -   if let;
    -   while let;
    -   for pattern;
    -   let pattern;
    -   func pattern;
- All match can occurred:
    -   match const;
    -   match var;
    -   match multi;
    -   match range;
- Match with deconstruction:
    -   deconstruct struct;
    -   deconstruct enum;
    -   deconstruct refer;
- Match & reference:
    -   ref;
    -   mut ref;
- Match guard;
- `@` bind;
