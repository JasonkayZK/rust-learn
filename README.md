## Create Project

Use Cargo to create a project:

```bash
cargo new hello_rust --bin
```

build:

```bash
cd hello_rust && cargo build --release
```

run:

```bash
./target/release/hello_rust
Hello, world!
```

>   Or just run program with:
>
>   ```bash
>   cargo run
>   ```
