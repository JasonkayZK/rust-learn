# Testing

- Write Tests
    - Define tests: `#[test]`
    - Panic in tests
    - Macros
        - `assert!`
        - `assert_eq!`
        - `assert_ne!`
        - `assert!` with `format!`
    - Check panic: `#[should_panic(expected="")]`
- Run Tests
    - Single/Multi thread tests: `-- --test-threads=1`
    - Show stdout from passed tests: `-- --nocapture`
    - Test specific testcase: `cargo test add_for_100`
    - Test part of testcases: `cargo test add`
    - Ignored tests:
        - `#[ignored]`
        - Test only ignored: `cargo test -- --ignored`
- Organize Tests
    - Unit Tests
        - Skip Compiling: `#[cgf(test)]`
        - Test private fn
    - Integration Tests
        - Directory `tests`
        - Test child mod
